# MT3608 -- Step-Up (Boost) DC-DC Converter

**Manufacturer:** Aerosemi Technology
**Package:** SOT23-6

## Device Overview

Fixed 1.2 MHz, current-mode step-up converter with integrated 80 mOhm N-channel MOSFET. Designed for small, low-power applications.

- Internal soft-start
- Auto PFM at light loads
- UVLO, current limiting (4A peak), thermal shutdown (155 C)
- Up to 97% efficiency

## Pin Descriptions

| Pin | Name | Function                                       |
| --- | ---- | ---------------------------------------------- |
| 1   | SW   | Power switch output (inductor + diode)         |
| 2   | GND  | Ground                                         |
| 3   | FB   | Feedback input (0.6V reference)                |
| 4   | EN   | Enable (high=on, low=off, tie to IN if unused) |
| 5   | IN   | Input supply                                   |
| 6   | NC   | No connection                                  |

## Output Voltage Formula

```
V_OUT = V_REF * (1 + R1 / R2)    where V_REF = 0.6V
```

R1: V_OUT to FB. R2: FB to GND.

| V_OUT | R2   | R1   |
| ----- | ---- | ---- |
| 3.3V  | 100k | 450k |
| 5V    | 100k | 733k |
| 12V   | 100k | 1.9M |

## Typical Application Circuit

```
        L1 (4.7-22uH)
VIN ---+---/\/\/---+---[D1]---+--- VOUT
       |           |          |
      [C1]      [SW]        [C2]
      22uF     [MT3608]     22uF
       |       [IN][EN]      |
      GND       |   |       GND
               GND  VIN
              [FB]---+
                     |
                    [R1] (to VOUT)
                     |
                    [R2] (to GND)
```

## Component Selection

- **Inductor:** 4.7-22 uH, low core loss at 1.2 MHz, saturation > 4A
- **Capacitors:** 22 uF ceramic (X5R/X7R), low ESR, close to IC
- **Diode:** Schottky, reverse voltage > V_OUT
- **Feedback resistors:** 1% tolerance, close to FB pin

## Efficiency

- **5V->12V:** peak ~93.5% at 200 mA, >88% from 100-500 mA
- **13V->18V:** 91-94% from 100-1700 mA
- Efficiency improves with lower boost ratios

## Layout

- Input/output caps close to IC on ground plane
- SW trace small to minimize EMI
- FB resistors close to FB pin, away from SW

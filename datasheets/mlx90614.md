# MLX90614 -- Infrared Non-Contact Thermometer

**Manufacturer:** Melexis
**Package:** TO-39

## Device Overview

Factory-calibrated infrared (IR) non-contact thermometer. Integrates an IR thermopile detector and signal-conditioning ASSP. Outputs linearized temperature via SMBus or PWM.

- **Ambient temperature range:** -40 to +125 C
- **Object temperature range:** -70 to +380 C
- **Resolution:** 0.02 C
- **Accuracy:** +/-0.5 C (0..+50 C), medical version +/-0.2 C (36..38 C)
- **Factory emissivity:** 1.0 (adjustable 0.1..1.0)
- **Variants:** single-zone (xAx), dual-zone (xBx), 5V (Axx), 3V (Bxx/Dxx)
- **Optical filter:** 5.5..14 um

## SMBus Protocol

- **Protocol:** SMBus (subset of I2C), device is always slave
- **Clock:** 10 kHz to 100 kHz
- **Default address:** 0x5A (7-bit), stored in EEPROM 0x0E
- **General call:** address 0x00 (all devices respond)
- **Data format:** 16-bit words (LSByte first) + 8-bit PEC (CRC-8, polynomial X8+X2+X1+1)
- **Commands:** Read Word and Write Word only

### Command Opcodes

| Opcode    | Command                                     |
| --------- | ------------------------------------------- |
| 0x00-0x1F | RAM access (bits [4:0] = address)           |
| 0x20-0x3F | EEPROM access (bits [4:0] = address)        |
| 0xF0      | Read flags                                  |
| 0xFF      | Enter SLEEP (3V only, PEC=0xE8 for SA=0x5A) |

### Read Word Sequence

```
S [SA_W] ACK [Command] ACK Sr [SA_R] ACK [LSB] ACK [MSB] ACK [PEC] NACK P
```

### Write Word Sequence

```
S [SA_W] ACK [Command] ACK [LSB] ACK [MSB] ACK [PEC] ACK P
```

### Timing

| Parameter                 | Value                |
| ------------------------- | -------------------- |
| SMBus request (t_REQ)     | >= 1.44 ms (SCL low) |
| Timeout low (SCL)         | <= 27 ms             |
| Wake-up request           | >= 33 ms (SCL low)   |
| Data available after wake | ~250 ms              |

## Memory Map

### RAM (read-only via SMBus)

| Address | Name       | Description                                   |
| ------- | ---------- | --------------------------------------------- |
| 0x06    | TA         | Linearized ambient temperature                |
| 0x07    | TOBJ1      | Linearized object temperature, sensor 1       |
| 0x08    | TOBJ2      | Object temperature, sensor 2 (dual-zone only) |
| 0x04    | Raw IR ch1 | Raw IR data                                   |
| 0x05    | Raw IR ch2 | Raw IR data                                   |

**Temperature conversion:** `T [C] = RAM_value * 0.02 - 273.15`

MSB (bit 15) is an error flag -- if set, reading is invalid.

### EEPROM (selected cells writable)

| Address   | Name              | Write | Description          |
| --------- | ----------------- | ----- | -------------------- |
| 0x04      | Emissivity        | Yes   | Default 0xFFFF = 1.0 |
| 0x05      | Config Register 1 | Yes   | Main configuration   |
| 0x0E      | SMBus address     | Yes   | LSB bits [6:0]       |
| 0x1C-0x1F | ID number         | No    | 64-bit unique ID     |

### EEPROM Write Procedure

1. Write 0x0000 to target cell (erase)
2. Wait >= 5 ms (10 ms recommended)
3. Read back and verify
4. Write new value
5. Wait >= 5 ms
6. Read back and verify
7. Power cycle

Endurance: 100,000 cycles. Retention: 10 years at 85 C.

## Emissivity Setting

- Formula: `EEPROM_value = round(65535 * emissivity)`
- Example: 0.95 -> 0xF332

## Configuration Register 1 (EEPROM 0x05)

| Bits    | Field           | Values                                                     |
| ------- | --------------- | ---------------------------------------------------------- |
| B0      | Self-test       | 0=enabled, 1=disabled                                      |
| B2-B4   | Amplifier gain  | 000=1, 001=3, 010=6, 011=12.5, 100=25, 101=50, 110/111=100 |
| B5-B7   | FIR filter      | 100=128, 101=256, 110=512, 111=1024                        |
| B8      | IR sensor count | 0=single, 1=dual                                           |
| B13-B15 | IIR filter      | 000=50%, 001=25%, 100=bypass                               |

**Do not modify bits [14:11, 7:3] without Melexis tools** -- cancels factory calibration.

## PEC (CRC-8) Calculation

- Polynomial: X^8 + X^2 + X^1 + 1 (0x107, init 0x00)
- Read: PEC over [SA_W, Command, SA_R, LSByte, MSByte]
- Write: PEC over [SA_W, Command, LSByte, MSByte]

## Initialization and Reading

### Minimal Init (SMBus, defaults)

No special init needed. After power-on:

1. Wait 250 ms for valid data
2. (Optional) Read flags (0xF0), check bit 3 = 1
3. Begin reading

### Reading Object Temperature

```
smbus_read_word(0x5A, 0x07) -> (lsb, msb, pec)
raw = (msb << 8) | lsb
if (raw & 0x8000) != 0 { error }
temp_celsius = raw * 0.02 - 273.15
```

### Sleep Mode (3V only)

- **Enter:** write command 0xFF (PEC=0xE8 for SA=0x5A)
- **Exit:** SCL low >= 33 ms, then wait 250 ms

## Application Notes

- 100 nF decoupling on VDD
- Pull-ups on SDA and SCL
- Continuous reads introduce noise; read less frequently than refresh rate
- VDD compensation (3V): `T_comp = T_meas - (VDD - 3.0) * 0.6` C/V
- Max 127 devices on one bus

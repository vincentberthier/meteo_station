# LiPo Battery (Seamuing 7649083)

3.7V 10000mAh lithium polymer battery with PH2.0A connector.

## Specifications

| Parameter              | Value             |
| ---------------------- | ----------------- |
| Voltage                | 3.7V              |
| Capacity               | 10000 mAh         |
| Max operating current  | 3A                |
| Overcurrent protection | 1.8-2.1A          |
| Dimensions             | 115 x 65 x 8.8 mm |
| Weight                 | 155g              |
| Connector              | PH2.0A            |

## Charging

- Constant current at 4.25V, 0.2C
- Then constant voltage at 4.2V until current <= 0.05C

Weather station “rooftop-01”

Terminal UI for a self-contained weather station. A summary band on top, a diagnostics bar kept separate from the data, then a 10-minute history. This document fixes the grid, palette, dynamic rules and a panel-by-panel reference for the Rust port.

Note — the interface ships in French; on-screen strings are quoted verbatim (e.g. « En direct »). All explanatory text here is English.

1Design principles

The screen reads top to bottom in four tiers of information, most immediate first:

  1. Header — station identity, clock, link state.
  2. Summary band — the instantaneous values grouped into three cards: Atmosphere, Wind (compass), Power.
  3. Diagnostics bar — hardware health, deliberately separated from the measurements.
  4. History — two 3-column grids: Sensors (6) and Power (3), 10-minute sliding window.

Cross-cutting rules:

  • Native terminal aesthetic. One monospace face (JetBrains Mono); bordered panels with the title inset into the top rule — Ratatui’s Block convention.
  • One hue per quantity. Each metric keeps its colour everywhere — value, plot, axis label. Colour carries meaning, never decoration.
  • “Braille” traces. Curves drawn as dotted point fields like Marker::Braille, with a faint gradient fill under the line (13 %→0).
  • Restrained chrome. Desaturated backgrounds, faint separators, layered greys; vivid colour is reserved for data and alerts.
  • Adaptive. The summary cards reflow; the history stays at 3 columns.

5Options & implementation notes

Exposed options

OptionEffect
markerStyledots (braille) or line (continuous) for all traces.
showGridShow / hide the horizontal dotted gridlines.
gustTrailShow / hide the compass gust trail.

Ratatui port

  • Panels: Block::default().borders(Borders::ALL).title(...); colours as Color::Rgb from the tables above.
  • Plots: Chart + Dataset with Marker::Braille, or a custom Canvas for the fill; axes via Axis with min/max labels.
  • Compass: Canvas (rings, ticks, trail points, needle); cardinals and centre readout as labels.
  • Refresh: redraw on each sample (1 Hz) or on event.
Note. The capture values are a sunny-afternoon scene, chosen to make the compass and power gauges legible. The interface strings are intentionally French; this spec is English. Thresholds (battery 50/20 %, RSSI −70/−90, packet 2/10 s) are starting points — tune to taste.
